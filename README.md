Project title:
  Build a Website on GenAI FAQs & Ethical Use of AI

Short overview:
  We are tasked to make a website that contains all about the commonly asked questions, ethical uses, and analogy of AI.

Instructions on how to run the project:
  1. Content Requirements

  A. GenAI FAQs
    What is Generative AI?
    How does it work?
    What are examples of GenAI tools?
    What can GenAI do (strengths)?
    What are its limitations or risks?
    Any other FAQs you want to add.
    References

  B. Ethical Use of AI
    Responsible use in the academe
    Avoiding plagiarism and misuse
    Accuracy and verification of AI outputs
    Bias and fairness
    Your own recommended guidelines
    References

  C. Analogy on the use of AI in Dev Work
    "You are organizing your wedding, the documentation team is asking 40k for the wedding package. They will use an Iphone 16 pro max to cover
    the event and CapCut for the same day edit video. The documentation team has 2 years of experience and has an above average portfolio. Would
    you agree to that deal? Explain your answer and justify the integrity of their work. Reflect on the integrity, creativity, skill level of the
    “vibe” coders."

  2. Website Requirements
    You must develop a static website using:
    Bootstrap (recommended) or Any frontend framework (e.g., React, Vue, Svelte, Tailwind, etc.)
    The site should have at least:
      Landing/Home page
      FAQ section
      Ethical Use of AI page
      Analogy on the use of AI in Dev Work
      Navigation menu

  Optional components (for better grade):
    Animations
    Icons
    Interactive UI elements (modals, tabs, accordions)

  Frameworks/Libraries Used:
    Bootstrap
    Technologies Used: HTML and CSS
